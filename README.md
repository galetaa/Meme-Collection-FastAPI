# API для мемов :)

Этот проект представляет собой веб-приложение, построенное с использованием FastAPI, которое предоставляет API для управления коллекцией мемов. Приложение состоит из двух сервисов: публичного API-сервиса с бизнес-логикой и медиа-сервиса для работы с медиа-файлами, использующего S3-совместимое хранилище (например, MinIO).

## Функциональность

- **GET /memes**: Получить список всех мемов (с пагинацией).
- **GET /memes/{id}**: Получить конкретный мем по его ID.
- **POST /memes**: Добавить новый мем (с изображением и текстом).
- **PUT /memes/{id}**: Обновить существующий мем.
- **DELETE /memes/{id}**: Удалить мем.

## Требования

- Docker
- Docker Compose

## Настройка

### 1. Клонируйте репозиторий

```bash
git clone <repository_url>
cd <repository_directory>
```

### 2. Постройка и запуск сервисов

```bash
docker-compose up --build
```

Эта команда запустит следующие сервисы:
- **db**: Сервис базы данных PostgreSQL
- **minio**: Сервис S3-совместимого хранилища MinIO
- **public-api**: Сервис публичного API
- **media-api**: Сервис медиа API

### 3. Запуск тестов

Для запуска тестов публичного API-сервиса:

```bash
docker-compose run --rm test-public-api
```

Для запуска тестов медиа API-сервиса:

```bash
docker-compose run --rm test-media-api
```

### 4. Доступ к приложению

- **Документация Public API**: [http://localhost:8000/docs](http://localhost:8000/docs)
- **Media API**: Доступен по адресу [http://localhost:8001](http://localhost:8001)
- **MinIO WebUI**: [http://localhost:9000](http://localhost:9000) (учетные данные по умолчанию: minioadmin/minioadmin)

## Переменные окружения

В проекте используются следующие переменные окружения:

- `DATABASE_USER`: Пользователь базы данных (по умолчанию: `memes_user`)
- `DATABASE_PASSWORD`: Пароль базы данных (по умолчанию: `password`)
- `DATABASE_HOST`: Хост базы данных (по умолчанию: `db`)
- `DATABASE_PORT`: Порт базы данных (по умолчанию: `5432`)
- `DATABASE_NAME`: Имя базы данных (по умолчанию: `memes`)
- `MINIO_HOST`: Хост MinIO (по умолчанию: `minio:9000`)
- `MINIO_ROOT_USER`: Корневой пользователь MinIO (по умолчанию: `minioadmin`)
- `MINIO_ROOT_PASSWORD`: Корневой пароль MinIO (по умолчанию: `minioadmin`)
